# Archiving Knowledge Base

## What this covers

How to use and maintain this `knowledge/` folder for Matomo archiving investigations.

## Key takeaways

- Start with `MAP.md` for the mental model and investigation order.
- Use `RECORD_BUILDERS.md` when the question is about plugin archive record generation behavior.
- Use `DATATABLES.md` for blob payload internals and DataTable-focused archive debugging.
- Keep all pages source-cited with exact file paths and URLs.
- Mark uncertainty explicitly as `Hypothesis`.

## Evidence

- `knowledge/MAP.md` links all topic pages and top source files.
